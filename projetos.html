Automatização de Planilhas

Desenvolvi um programa em Python para resolver um problema enfrentado por uma contadora. Ela tinha uma planilha com vários códigos referentes a tarefas, e minha solução foi criar um script que lê o Excel original e adiciona uma nova coluna com esses códigos. Isso simplificou significativamente o processo manual e economizou tempo para essa contadora.

Controle de Estoque em Java

Desenvolvi um projeto em Java para treinar minha lógica de programação, implementando um sistema de controle de estoque. Utilizei arrays e listas para armazenar os dados dos produtos e criei funcionalidades básicas, como adicionar, atualizar e remover produtos.

Color Flipper

Desenvolvi um projeto de "Color Flipper" para praticar minhas habilidades em HTML, CSS e JavaScript. Este projeto consiste em uma aplicação web simples que gera cores aleatórias ao clicar em um botão. A cor de fundo da página muda para a cor gerada, proporcionando uma experiência visual dinâmica e divertida.

Utilizei HTML para estruturar a página, CSS para estilizar o layout e JavaScript para implementar a lógica de geração de cores aleatórias. A interação com o usuário é facilitada por meio de um botão que, ao ser clicado, aciona a mudança de cor.

Este projeto me permitiu consolidar meus conhecimentos em manipulação do DOM (Document Object Model), eventos e estilos dinâmicos, além de proporcionar uma introdução prática ao desenvolvimento front-end.
